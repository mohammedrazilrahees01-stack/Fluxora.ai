# Fluxora

Premium AI developer automation platform built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Dark luxury UI
- Glassmorphism design
- Animated backgrounds
- Dashboard
- AI Chat workspace
- Projects management
- Billing page
- Settings page
- Login / Register pages
- Reusable components

## Routes

- `/`
- `/dashboard`
- `/chat`
- `/projects`
- `/billing`
- `/settings`
- `/pricing`
- `/login`
- `/register`
